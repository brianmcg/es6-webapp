# ES6 Webapp

This is a skeleton webapp built with [Webpack](https://github.com/webpack) and [Materialize](https://materializecss.com/)

## Getting Started

### Install
```
npm install
```

### Build
```
npm run build
```

### Prerequisites

Created using [npm]() 10.4.0
